# 本地化文档

## 目标

StillLoop v1 支持简体中文和英文。应用语言跟随 macOS 系统语言，不提供应用内语言切换，也不保存独立语言偏好。

## 范围

本轮本地化覆盖固定用户界面文案：

- 欢迎、权限、模型准备、任务输入、专注运行、复盘、设置、隐私、开源许可页面。
- 菜单栏标题和菜单项。
- 权限、模型下载、模型连接、运行状态、toast 和系统设置引导文案。
- 应用代码生成的 nudge 文案。

以下内容不在本轮多语言范围内：

- 模型生成的 reason、评语或其他自然语言输出。
- LLM prompt 内容。
- 已保存历史会话中的 nudge、reason 和调试文本。
- App Store metadata、营销截图和官网文案。

## 行为规则

- 用户输入的任务、App 名称、窗口标题、浏览器标题和 URL 摘要保持原文。
- telemetry event name、raw value、持久化字段和诊断字段名保持稳定，不本地化。
- 业务判断不能依赖中文或英文展示文案；权限、模型运行和分析进度必须使用结构化状态。
- 本地化资源使用语义 key，中文是默认语言，英文提供完整翻译。

## 验收标准

- `zh-Hans` 和 `en` 下关键 copy key 都能返回非空文本。
- 英文环境下，权限缺失、模型下载中、模型运算中和 nudge 展示不依赖中文字符串。
- 固定 UI 英文文案不明显溢出主要卡片、按钮和菜单栏。
